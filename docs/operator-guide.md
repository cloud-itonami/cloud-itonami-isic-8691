# Operator Guide

## First Deployment

1. Build a provider directory.
2. Define consent and purpose categories.
3. Run synthetic intake cases.
4. Enable human-reviewed referrals.
5. Measure referral completion and access gaps.

## Minimum Production Controls

- consent and disclosure log
- urgent-risk escalation path
- provider directory provenance
- human review for sensitive cases
- audit export for referrals

## Certification

Certified operators must prove that the system is a navigation and referral
tool, not a diagnostic authority, and that urgent risks escalate to humans.
